# 10 Pure Javascript Tools for any project that are lightwright and multipurpose

> Here is the top 10 tools i think are good alrounder for any projects and more mostly vanlla JS. You won't see no jquery here (inshallah)

1. [Greensock Animation Framework - Extremely powerful](https://greensock.com/gsap/) 

2. [Plyr HTML5 video player](https://plyr.io/)

3. [Dropzone File upload tool](https://www.dropzonejs.com/)

4. [Flat Picker Calendar Picker- Great flexible calendar picker, cuz we all need one](https://flatpickr.js.org/)

5. [Cleave.js a - brilliant number format, also has integeration into react as a prebuilt component](https://nosir.github.io/cleave.js/)

6. [Countup.js - a simple single purpose tool to create counting animation in Pure js](https://inorganik.github.io/countUp.js/)

7. [Rellax - A pure javascript tool for creating parallax website / animation easily](https://dixonandmoe.com/rellax/)

8. [Particles.js - powerful particle animation library](https://vincentgarreau.com/particles.js/)

9. [Smooth Scrollbar - Smooth ios style scrolling with tons of browser compatability in pure js ](https://idiotwu.github.io/smooth-scrollbar/)

10. [filterizr - JS and CSS3 filtering system for usage on galleries](https://yiotis.net/filterizr/#/)
